# Code of Conduct

This project follows the spirit of the Contributor Covenant. Be respectful, inclusive, and constructive. Harassment, discrimination, personal attacks, and spam are not tolerated.

Reporting

- For conduct concerns, open an issue (non-sensitive) or contact the maintainer privately via GitHub.
- The maintainer may moderate content and restrict participation when behavior violates this policy.
